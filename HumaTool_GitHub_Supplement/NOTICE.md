# Notice

This repository is prepared as a public-safe academic project package.

The following materials are intentionally not included:

- raw teleoperation demonstrations
- private robot logs
- model checkpoints and pretrained weights
- hardware IP addresses, serial numbers, tokens, or local file paths
- proprietary SDK files

Before making this repository public, please confirm that all figures, source code, datasets, and third-party dependencies are allowed to be shared by the project team and the relevant laboratory.
