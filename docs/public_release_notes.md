# Public Release Notes

This repository is prepared as a public academic project showcase. It is suitable for portfolio review, advisor communication, and high-level reproduction of the project workflow.

## Included

- Project overview and portfolio PDF
- README figures and system diagrams
- RoboNeck-AVP-related documentation
- Public-safe configuration examples
- Demonstration data schema
- Data processing and VLA fine-tuning templates
- Citation and notice files

## Excluded

- Raw demonstration trajectories
- Full-resolution video recordings
- pretrained and fine-tuned model checkpoints
- private robot configuration files
- local device paths, IP addresses, and serial numbers
- third-party SDK source code

## Recommended Repository Visibility

Before confirming with the instructor or lab, keep the repository private. After approval, it can be made public as a project showcase.
